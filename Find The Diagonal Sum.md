# Find The Diagonal Sum

Fork this repository and solve the following problem.
---

Given a square matrix of order N ∗ N, your task is to find the sum of the elements of the primary and secondary diagonal of the matrix.

Note: The element common to both the diagonals should only be counted once.

## Input
The first line contains an integer N representing the size of the matrix.  
The next N lines contain N space-separated integers representing the matrix.

Example Input:
3  
1 2 3  
4 5 6  
7 8 9  

## Output
Print a single integer representing the sum of the primary and secondary diagonals.

Example Output:
25
