# Check Arithmetic Progression of Four Non Decreasing Numbers

Fork this repository and solve the following problem.
---

You are given four integers x1, x2, x3, x4 in non-decreasing order. Your task is to check whether these four numbers form an arithmetic progression (AP) or not.

Formally, a sequence x1, x2, …, xn is called an arithmetic progression (AP) if there exists a number d (called the common difference) such that: xi+1 = xi + d for all i from 1 to n - 1.

A sequence is in non-decreasing order if each element is greater than or equal to the previous one. For example: 1 2 2 3 and -5 -5 0 7 are in non-decreasing order, but 4 2 6 8 is not.

## Input
The input consists of a single line containing four space-separated integers:
x1, x2, x3, x4

Example Input:
2 4 6 8

## Output
Print a single word:  
YES if the four numbers form an arithmetic progression  
NO otherwise.

Example Output:
YES
